# 點餐bot

## 目前接受指令

聽取使用說明:

`說明/`

獲取餐廳列表:

`餐廳/`

決定今天吃的餐廳:

`吃/[餐廳名稱]`

點餐:

`點/[餐點編號1]/[餐點編號2]/...`

取消點餐:

`取消/[餐點編號1]/[餐點編號2]/...`

統計數量、金額、生成明細表

`統計/`

關閉點餐:

`截止/`

清除今日的訂餐資料:

`clear/`
